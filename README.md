# Intelligent Sanction Analysis
*Using Knowledge graphs and Small Language Models* 

<img align="right" alt="Coding" width="550" src="https://github.com/user-attachments/assets/f7b154d0-4b24-47dc-ae50-e593cfff36bd">

This project presents the implementation of an intelligent sanction analysis system based on Knowledge Graphs and Small Language Models. By integrating structured sanction data derived from converting 
INTERPOL Red Notices into a graph-based representation, the system enables efficient, explainable, and 
scalable analysis of complex relational information.
The combination of symbolic reasoning through graph queries and language-based reasoning using a 
A small language model allows the system to provide transparent and meaningful answers to analytical 
questions. This approach demonstrates the relevance of hybrid AI architectures for sensitive domains 
such as law enforcement, compliance, and intelligence analysis
